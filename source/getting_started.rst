===============
Getting Started
===============

To start using Citrix Cloud APIs, you must:

1) If you haven't already, `sign up for a Citrix Cloud account
   <https://onboarding.cloud.com>`_.
2) `Create an API client <create_api_client.html>`_.
3) `Authenticate using the API client <authenticate_api_client.html>`_, to get a
   Citrix Cloud bearer token.
4) `Call the API <call_api_bearer_token.html>`_, passing the bearer token as the
   ``Authentication`` header in the request.
